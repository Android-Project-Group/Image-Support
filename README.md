# ImageBadgeView
Library to add ImageView (ImageBadgeView) with a badge like notification count.
## Download
Add to gradle root:
```
buildscript {
    repositories {
        jcenter()
    }
```
Download via Gradle:
```
compile 'com.github.nikartm:image-support:1.0.0'
```
or Maven:
```
<dependency>
  <groupId>com.github.nikartm</groupId>
  <artifactId>image-support</artifactId>
  <version>1.0.0</version>
  <type>pom</type>
</dependency>
```
